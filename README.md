## ✨ Overview

A beautiful, responsive one-page portfolio website designed for **UI/UX Product Designers**. This portfolio showcases design work and projects with a focus on:

- **Stakeholder Alignment** - Bridging business needs with user insights
- **User Research** - Data-driven design decisions
- **Product Experience** - Clear, comfortable, and scalable product experiences

## 🌟 Features

### 🎨 Design Elements
- ✅ **Dark/Light Theme Toggle** - Seamless theme switching with smooth transitions
- ✅ **Liquid Animation** - Dynamic hero canvas with liquid effects
- ✅ **Scroll Progress Indicator** - Visual progress bar for page navigation
- ✅ **Cursor Glow Effect** - Interactive cursor hover effects
- ✅ **Smooth Scroll Snap** - Smooth section transitions

### 📱 Responsive Design
- ✅ **Mobile-First Approach** - Fully responsive across all devices
- ✅ **Modern Typography** - Beautiful fonts with perfect readability
- ✅ **Accessibility** - ARIA labels and semantic HTML

### 🚀 Performance
- ✅ **Optimized Assets** - Fast loading with minimal dependencies
- ✅ **CSS Animations** - Smooth, performant transitions
- ✅ **Clean Code** - Well-structured, maintainable codebase

## 📦 Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/adamramadhn/aldin-portfolio.git
   cd aldin-portfolio
   ```

2. **Open the project**
   ```bash
   # Simply open index.html in your browser
   open index.html
   ```

3. **Customize content**
   - Edit `index.html` to update your name, projects, and content
   - Modify `styles.css` to adjust colors, fonts, and styling
   - Update `script.js` for custom interactions

## 🚀 Deployment

### GitHub Pages
This project is hosted on GitHub Pages with a custom domain:

**🌐 Live Demo:** https://aldin.theltsoul.my.id

### Custom Domain Setup
1. Add a `CNAME` file with your domain:
   ```
   your-subdomain.yourdomain.com
   ```

2. Configure your domain's DNS:
   - Add a CNAME record pointing to `your-username.github.io`

3. Enable GitHub Pages in repository settings

## 🎨 Customization

### Update Your Information
Edit `index.html` to personalize:
- Your name and title
- Project descriptions and images
- Contact information
- Social media links

### Theme Colors
Modify CSS variables in `styles.css`:
```css
:root {
    --primary-color: #3B82F6;
    --text-color: #1F2937;
    --bg-color: #FFFFFF;
    /* ... more variables */
}
```

### Add Projects
Follow the project card structure in the HTML:
```html
<div class="project-card">
    <h3>Project Title</h3>
    <p>Project description...</p>
</div>
```
